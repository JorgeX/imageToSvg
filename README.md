# Image to SVG Converter

A simple web application that converts raster images (PNG, JPG) into scalable vector graphics (SVG) using ImageTracerJS.

## Features

- Drag and drop or click to upload images
- Adjustable conversion settings:
  - Number of colors (2-32)
  - Blur radius
  - Curve detail
- Live preview of the original image and SVG output
- Download the converted SVG file
- Dark mode support
- Responsive design that works on mobile and desktop

## How to Use

1. Click "Upload Image" or drag and drop an image file
2. Adjust the conversion settings as needed
3. Click "Convert to SVG"
4. Preview the result
5. Click "Download SVG" to save the converted file

## Technologies Used

- HTML5, CSS3, JavaScript
- [ImageTracerJS](https://github.com/jankovicsandras/imagetracerjs) for image tracing
- [Tailwind CSS](https://tailwindcss.com/) for styling
- [Inter](https://rsms.me/inter/) font

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge)
